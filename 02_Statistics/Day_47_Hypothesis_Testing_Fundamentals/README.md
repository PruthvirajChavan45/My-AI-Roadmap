## Hypothesis Testing
Hypothesis testing is a statistical decision-making framework used to test claims about a population using sample data.  
We start by assuming a default statement is true and then use evidence from data to decide whether to keep or reject it.

### Types of Hypothesis
- **Null Hypothesis (H‚ÇÄ):**  
  Represents ‚Äúno effect‚Äù, ‚Äúno difference‚Äù, or the current assumption.
- **Alternative Hypothesis (H‚ÇÅ):**  
  Represents a change, effect, or difference that challenges H‚ÇÄ.

The process always begins by assuming H‚ÇÄ is true and then checking whether the data strongly contradicts it.

---

## Significance Level (Œ±)
The significance level is the cut-off value used to decide whether to reject the null hypothesis.
- Common value: **Œ± = 0.05**
- It represents the probability of making a wrong decision while rejecting H‚ÇÄ.
- Œ± = 0.05 means we accept a 5% risk of being wrong.

---

## Type of Errors
When making decisions in hypothesis testing, two kinds of errors can occur:

- **Type I Error:**  
  Rejecting H‚ÇÄ even though it is actually true (false alarm).
- **Type II Error:**  
  Failing to reject H‚ÇÄ even though it is actually false (missed detection).

---

## One-Tailed and Two-Tailed Tests
- **One-Tailed Test:**  
  Used when we care about a change in only one direction (greater than or less than).
- **Two-Tailed Test:**  
  Used when we care about any difference, regardless of direction.

---

## P-Value
The p-value measures how extreme the observed data is assuming H‚ÇÄ is true.
- Smaller p-value ‚Üí stronger evidence against H‚ÇÄ.
- **Decision Rule:**
  - If `p-value < Œ±` ‚Üí Reject H‚ÇÄ
  - If `p-value ‚â• Œ±` ‚Üí Fail to reject H‚ÇÄ

---

## Test Statistic
A test statistic is a numerical value calculated from sample data that helps decide whether to reject H‚ÇÄ.
It shows how far the sample result is from what we would expect if H‚ÇÄ were true.

Common examples:
- Z-test  
- t-test  
- Chi-square test  
- F-test
